API Reference

All endpoints return JSON. Free endpoints need no authentication. Paid endpoints use the x402 payment protocol.

FREE   List Available Domains

Returns all domains with cached backlink data, including backlink counts and crawl dates.

curl https://backlink-finder.fly.dev/domains # Response: { "domains": [{ "domain": "tesla.com", "backlink_count": 30542, ... }], "total": 19 }

FREE   Preview Backlinks

Returns top 5 notable backlinks for any cached domain. Great for evaluating data quality before paying.

curl https://backlink-finder.fly.dev/preview/tesla.com # Response: { "domain": "tesla.com", "total_backlinks": 30542, "showing": 5, # "backlinks": [{ "linking_domain": "alibaba.com", "num_hosts": 86199, "authority_score": 74 }, ...] }

$0.10   Full Backlink Lookup

Returns all linking domains for any domain. Crawls on-demand from web data if not cached (2–5 min). AI agents pay automatically via x402.

curl https://backlink-finder.fly.dev/backlinks/example.com # Without payment → 402 response with x402 payment details # With x402 payment → { "domain": "...", "backlink_count": N, "backlinks": [{"linking_domain": "...", "authority_score": 74, ...}] }

$0.15   Gap Analysis

Compare two domains — find sites linking to a competitor but not to you. Both domains must be cached.

curl "https://backlink-finder.fly.dev/gap?yours=you.com&competitor=them.com" # Response: { "gap_count": 847, "opportunities": [{ "domain": "techcrunch.com", ... }] }

DISCOVERY   Agent Integration

AI agents discover endpoints, pricing, and payment details automatically via the x402 well-known file.

curl https://backlink-finder.fly.dev/.well-known/x402.json # Returns: payment network, wallet address, per-endpoint pricing, and schemas
