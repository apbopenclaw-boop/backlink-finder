Frequently Asked Questions

What is bhrefs?

bhrefs is a suite of SEO tools built specifically for AI agents. Tools include backlink discovery, gap analysis, and more — all accessible via API with pay-per-query USDC micropayments through the x402 protocol. No API keys, no accounts, no subscriptions.

How do AI agents use bhrefs?

AI agents make standard HTTP requests to bhrefs API endpoints. The API returns a 402 status with payment details. The agent pays with USDC on the Base network, and receives SEO data in the same request cycle — fully autonomous.

What is the x402 payment protocol?

x402 is an open protocol that lets AI agents pay for API access using cryptocurrency micropayments. When an agent hits a paid endpoint, it receives a 402 Payment Required response with pricing, pays with USDC, and gets data — all programmatically.

How much does bhrefs cost?

Pay-per-query with no subscriptions. Backlink lookups: $0.10 USDC. Gap analysis: $0.15 USDC. Preview lookups: free. Payments in USDC on the Base network with sub-cent gas fees.

What backlink data does bhrefs provide?

bhrefs Backlink Finder scans billions of hyperlinks from the Common Crawl web archive. Data includes linking domains, host counts, and authority scores. On-demand crawling available for uncached domains (2–5 minutes).

Do I need an API key to use bhrefs?

No. bhrefs is completely anonymous — no accounts, no API keys, no registration. Any AI agent with a USDC wallet on Base can query instantly. Free endpoints require nothing at all.
